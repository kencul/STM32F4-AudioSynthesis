The Csound API
========

Csound API public headers

- arrays.h: array creation and initialisation
- cfgvar.h: configuration variables for extra engine options
- coreDefs.h: core system definitions
- csPerfThread.h: csound performance thread C interface
- csPerfThread.hpp: csound performance thread C++ interface
- cs_par_structs.c: multicore functionality
- csdebug.h: csound debugger interface
- csdl.h: csound modules and plugins header
- csound.h: main system API header
- csoundCore.h: core system functionality and module API
- csound_circular_buffer.h: circular buffer interface
- csound_compiler.h: lower-level parsing and compiling interface
- csound_data_structures.h: data structures support
- csound_files.h: file access interface
- csound_graph_display.h: graphs and display interface
- csound_misc.h: miscellaneous API functions
- csound_rtaudio.h: realtime audio interface
- csound_rtmidi.h: realtime MIDI interface
- csound_server.h: UDP server interface
- csound_threads.h: threading interface
- csound_type_system.h: type system interface
- cwindow.h: window graphs functionality
- float-version.h.in: source file for float-version.h (setting MYFLT size)
- interlocks.h: locks interface
- modload.h: plugin loading 
- msg_attr.h: message printing attributes
- plugin.h: csound plugin opcode framework
- pstream.h: streaming spectral processing structs
- pvfileio.h: PVOCEX file support
- soundfile.h: soundfile IO interface
- soundio.h: sound IO definitions
- sysdep.h: system dependencies
- text.h: internationalisation support
- ugen.h: ugen API
- version.h.in: source file for version.h (setting version constants)
